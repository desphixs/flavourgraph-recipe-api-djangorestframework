# Build FlavorGraph: A Smart Recipe API

### Project Description

You are going to build FlavorGraph, a back-end engine for a smart recipe app. You will start by setting up a library of individual ingredients and cuisines. Once your building blocks are ready, you will build the core feature: recipes that combine multiple ingredients together. You will write custom code to show the full details of every ingredient inside a recipe, build a search tool so users can find meals based on what is in their fridge, and write logic to page through long lists of recipes. Finally, you will add a review section so users can share cooking tips. You will build all of this using raw APIView classes so you understand every single detail of how requests and database relationships work under the hood.

## Tech Stack

- Django
- Django REST Framework
